## Install requirements
1. PHP5.6
**How to list PHP versions installed**
    ```
    sudo update-alternatives --list php
    ```
    **OR**
    ```
    cd /etc/php
    dir
    ```
**Switching between php**
    ```
    sudo update-alternatives --config php
    ```
